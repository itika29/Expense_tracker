
💸 Expense Tracker (CLI Based)

A simple and efficient Command-Line Expense Tracker built using Python.

This project started as a dictionary-based expense manager and is being progressively upgraded with better structure and database integration.

🚀 Features (Version 1 – Dictionary Based)

📌 Set a total budget

🗂 Predefined expense categories:

Food

Travel

Stationary

Miscellaneous

➕ Add expenses category-wise

📊 Display all expenses

💰 Calculate remaining budget

🧠 Clean modular structure using functions

🛠 Tech Stack

Python 3

CLI (Command Line Interface)

Modular programming approach
📌 Future Improvements

SQLite database integration

Expense search functionality

Monthly summary report

Input validation

GUI or Web App version

💡 How To Run

Clone the repository:

git clone <your-repo-link>

Navigate into the folder:

cd Expense-Tracker

Run the program:

python main.py
